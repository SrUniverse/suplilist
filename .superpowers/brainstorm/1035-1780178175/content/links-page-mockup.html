Como funciona o funil
📸 No Instagram

Bio: suplilist.com/links?utm_source=instagram&utm_medium=bio

Cada post pode usar um link específico:
/links?utm_campaign=creatina

📊 No Plausible

Você verá:
• Quantas pessoas vieram da bio vs. de posts
• Qual objetivo elas escolheram
• Qual campanha converteu mais usuários

⚡ O que acontece ao clicar

Usuário clica "Hipertrofia" →
Vai para /list?objective=Hipertrofia&utm_source=instagram

Catálogo já abre filtrado pelo objetivo. Plausible registra a conversão.

🗓 Exemplos de uso por campanha
Post sobre creatina → bio link com utm_campaign=creatina
Story "suplementos para dormir" → /list?objective=Longevidade&utm_campaign=sono
Reels comparando creatinas → /links?utm_campaign=reels-creatina
