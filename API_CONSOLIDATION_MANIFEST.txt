RESPONSE FORMAT REFERENCE
================================================================================

SUCCESS RESPONSE (200/201/etc):
{
  "success": true,
  "data": { ... },
  "meta": {
    "timestamp": "2024-01-01T12:00:00Z",
    "pagination": { "total": 100, "page": 1, "limit": 10, "hasMore": true },
    "requestId": "req_123"
  }
}

ERROR RESPONSE (400/404/500/etc):
{
  "success": false,
  "error": {
    "code": "not_found",
    "message": "Resource not found",
    "details": { "field": "Error message" }
  },
  "meta": {
    "timestamp": "2024-01-01T12:00:00Z",
    "requestId": "req_123"
  }
}

DEPRECATED RESPONSE (v1 endpoint):
{
  "success": true,
  "data": { ... },
  "meta": { ... },
  "deprecation": {
    "message": "Use /api/v2/... instead",
    "sunset": "2024-12-01T00:00:00Z",
    "alternative": "/api/v2/endpoint"
  }
}

Headers on deprecated endpoints:
  Deprecation: true
  Sunset: 2024-12-01T00:00:00Z
  Link: </api/v2/endpoint>; rel="successor-version"
