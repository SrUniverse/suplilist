## 🚀 Próximos Passos (5 minutos)

### 1️⃣ Adicionar Secret no GitHub

```
GitHub → seu repo → Settings → Secrets and variables → Actions
```

Clique **New repository secret**:
```
Nome: FIRECRAWL_API_KEY
Valor: FIRECRAWL_KEY_REMOVED
```

✅ Save

### 2️⃣ Fazer Commit

```bash
cd ~/Desktop/suplilist

git status
# Você vê: .github/workflows, scripts/, package.json

git add .
git commit -m "chore: add GitHub Actions MCP automation

- Firecrawl: monitor preços Shopee, Mercado Livre, Amazon
- Chrome DevTools: auditoria de performance diária
- Playwright: E2E tests de fluxos críticos
- Schedule: domingo 9am, diariamente 6am, seg-sex 2pm (UTC)"

git push
```

### 3️⃣ Testar

```bash
# Ver se workflow apareceu
gh workflow list

# Disparar manualmente
gh workflow run mcp-automation.yml

# Verificar resultado (aguarde 2-3 min)
gh run list
gh run view <RUN_ID> --log
```

## 🆘 Troubleshooting Rápido

| Problema | Fix |
|----------|-----|
| "Secret invalid" | Copiar API key correto do Firecrawl |
| "Workflow not found" | Refresh page GitHub, ou aguardar 1min |
| "cron not triggered" | Validar syntax em https://crontab.guru |
| "Scripts not found" | Checar se `.github/` e `scripts/` foram pushed |
