# Firecrawl Automation — Weekly Price Updates

Como automatizar o workflow de monitoramento de preços para rodar **toda semana** sem intervenção manual.

## 3 Opções de Automação

### Opção 1: GitHub Actions (Recomendado)

Roda o workflow toda semana via CI/CD, atualiza `affiliate.config.js` e faz commit automático.

**Setup (5 minutos):**

1. Criar `.github/workflows/firecrawl-weekly-prices.yml`:

```yaml
name: Firecrawl Weekly Price Monitor

on:
  schedule:
    - cron: '0 9 * * 1'  # Segunda 9am UTC
  workflow_dispatch:     # Manual trigger também

jobs:
  monitor-prices:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      
      - name: Setup Node
        uses: actions/setup-node@v3
        with:
          node-version: '24.15.0'
      
      - name: Run Firecrawl workflow
        env:
          FIRECRAWL_API_KEY: ${{ secrets.FIRECRAWL_API_KEY }}
        run: |
          npx -y claude-code run --workflow suplilist-price-monitor
      
      - name: Update affiliate.config.js
        run: |
          # Script para atualizar affiliate.config.js com dados do workflow
          npm run scripts/update-prices.js
      
      - name: Commit if changed
        run: |
          git config user.name "Price Monitor Bot"
          git config user.email "bot@suplilist.com"
          git add src/monetization/affiliate.config.js
          git diff --cached --exit-code || git commit -m "chore: update supplement prices via Firecrawl"
          git push
```

2. Adicionar `FIRECRAWL_API_KEY` aos GitHub Secrets:
   ```
   Settings → Secrets → New secret
   Name: FIRECRAWL_API_KEY
   Value: fc-b8dc5832f0c94788bb177acfbbc07199
   ```

3. Script para atualizar preços (`scripts/update-prices.js`):

```javascript
#!/usr/bin/env node

const fs = require('fs');
const path = require('path');

// Simular leitura de workflow output
const workflowOutput = JSON.parse(process.env.WORKFLOW_OUTPUT);
const { consolidated_prices } = workflowOutput;

// Gerar código
const code = `
// Generated by Firecrawl workflow — DO NOT EDIT MANUALLY
export const SUPPLEMENT_PRICES = ${JSON.stringify(consolidated_prices, null, 2)};

// Updated: ${new Date().toISOString()}
`;

// Escrever arquivo
const configPath = path.join(__dirname, '../src/monetization/affiliate.config.js');
fs.writeFileSync(configPath, code, 'utf8');
console.log('✅ Updated affiliate.config.js');
```

### Opção 2: Claude Code Scheduled Task

Roda o workflow em um horário específico usando Claude Code's task scheduling.

```bash
# Criar task semanal
create_scheduled_task(
  taskId: 'suplilist-firecrawl-prices',
  prompt: 'Run the suplilist-price-monitor workflow and update affiliate.config.js',
  cronExpression: '0 9 * * 1',  # Segunda 9am
  notifyOnCompletion: true
)
```

**Vantagem**: Integrado ao seu workflow de Claude Code

### Opção 3: Cron Job Local

Para rodar localmente em seu computador.

```bash
# Adicionar ao crontab
(crontab -l 2>/dev/null; echo "0 9 * * 1 cd /path/to/suplilist && npm run firecrawl:monitor") | crontab -

# Verificar
crontab -l
```

Create `package.json` script:

```json
{
  "scripts": {
    "firecrawl:monitor": "node scripts/firecrawl-monitor.js"
  }
}
```

Script `scripts/firecrawl-monitor.js`:

```javascript
#!/usr/bin/env node

const { execSync } = require('child_process');

try {
  console.log('🚀 Starting Firecrawl price monitor...');
  
  // Run Claude Code workflow
  const result = execSync('npx claude-code run --workflow suplilist-price-monitor', {
    stdio: 'inherit'
  });
  
  // Update affiliate.config.js
  execSync('npm run scripts/update-prices.js', { stdio: 'inherit' });
  
  // Commit
  execSync('git add src/monetization/affiliate.config.js', { stdio: 'inherit' });
  execSync('git commit -m "chore: update supplement prices"', { stdio: 'inherit' });
  
  console.log('✅ Price monitor completed successfully');
} catch (error) {
  console.error('❌ Price monitor failed:', error.message);
  process.exit(1);
}
```

## Recomendação

**Use Opção 1 (GitHub Actions)** porque:
- ✅ Roda sem requerer seu computador ligado
- ✅ Histórico de execução em GitHub
- ✅ Fácil de debugar (logs públicos)
- ✅ Integra com CI/CD existente
- ✅ Auto-commit e push

## Notificações

Configurar alertas quando preços mudam significativamente:

```javascript
// No workflow: após consolidation
const significantChanges = Object.entries(consolidated_prices)
  .filter(([id, data]) => {
    const oldPrice = PREVIOUS_PRICES[id];
    const newPrice = data.shopee?.price;
    const percentChange = Math.abs((newPrice - oldPrice) / oldPrice * 100);
    return percentChange > 20; // 20% change threshold
  });

if (significantChanges.length > 0) {
  // Send notification (email, Slack, etc)
  notifyPriceAlert(significantChanges);
}
```

## Monitoramento

Para acompanhar execuções:

```bash
# Ver histórico no GitHub Actions
gh run list --workflow=firecrawl-weekly-prices.yml

# Ver último run
gh run view --latest --workflow=firecrawl-weekly-prices.yml

# Ver logs detalhados
gh run view <RUN_ID> --log
```

## Troubleshooting

| Problema | Solução |
|----------|---------|
| Workflow não rodou | Verificar GitHub Actions logs ou cron logs |
| Preços não atualizaram | Validar FIRECRAWL_API_KEY nos secrets |
| Commit falhou | Verificar se branch está protegido |
| Rate limit | Reduzir frequência ou usar proxy |
