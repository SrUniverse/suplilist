- batchSize: number (default: 10)
- batchInterval: number (default: 30000ms)
- maxErrors: number (default: 100)
- captureConsoleError: boolean (default: true)
- excludePatterns: RegExp[] (filters 404, 403, test errors)

BACKEND INTEGRATION REQUIRED
================================================================================

Endpoint: POST /api/logs/errors
Payload:
{
  "errors": [
    {
      "type": "UNCAUGHT_ERROR|UNHANDLED_REJECTION|CONSOLE_ERROR|COMPONENT_ERROR",
      "message": string,
      "stack": string,
      "context": object,
      "trace_id": string,
      "user_id": string | null,
      "timestamp": ISO8601,
      "url": string,
      "userAgent": string
    }
  ]
}

Endpoint: POST /api/metrics/performance
Payload:
{
  "metric": "lcp|fid|cls|page_load_time|api_*",
  "value": number,
  "timestamp": ISO8601,
  "url": string
}
